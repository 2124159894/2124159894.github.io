1.CSS3基本概念

1.1CSS3是什么

CSS的全称是Cascading Style Sheets，层叠样式表，一种标记语言，负责展示页面样式。

1.2.CSS3能做什么

主要是把页面的结构和样式分离，专门负责样式展示。

1.3.一个简单的CSS3示例

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <style>
            div{
                background-color:red;
            }
        </style>
    </head>
    <body>
        <div>
            Hello World
        </div>
    </body>
</html>
